## ğŸ“ Image Naming Rules

1. **All lowercase**: `apple.png` not `Apple.png`
2. **Hyphens for spaces**: `french-fries.png` not `french fries.png`
3. **PNG format**: For transparency support
4. **Descriptive names**: Use English name for easy identification
